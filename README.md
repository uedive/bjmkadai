# 実行方法
1. 下準備
```
npm install
```
2. APIキーの設定
.env をルートフォルダに作成（.env_sampleをコピー）
以下のようにAPIKEYを記載
```
REACT_APP_API_KEY="ここにAPIKEYを入力"
```
